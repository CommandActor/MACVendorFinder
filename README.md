# MACVendorFinder
Simple tool that can find the vendor of a MAC.

macven.py -> Usage: python3 macven.py [mac_address]

macvenls.py -> Reads MACs from a file. Usage: python3 macvenls.py [maclist.txt]
               Example maclist.txt:
               00:00:00:00:00:00
               00:00:00:00:00:01
               00:00:00:00:00:02
               00:00:00:00:00:03
               00:00:00:00:00:03
               etc...
